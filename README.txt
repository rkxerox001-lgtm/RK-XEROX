# RK Xerox Website
Single-file static website for RK Xerox.

## Quick Deploy (GitHub Pages)
1) Create repo `rkxerox.github.io` (Public).
2) Upload `index.html` to the root.
3) Settings → Pages → Source: Deploy from a branch → Branch: main → Save.
4) Live at https://rkxerox.github.io

## Quick Deploy (Netlify)
1) Go to https://app.netlify.com → Add new site → Deploy manually.
2) Drag & drop this folder (or just index.html).
3) Get the live URL instantly.
